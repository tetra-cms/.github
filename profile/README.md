# <img width="32" height="32" src="https://github.com/user-attachments/assets/df5b3e07-38a4-40a7-8483-f1fed8338c7d"> TETRA 
Open-source simplified and flexiable [**E-Commerce CMS**](https://en.wikipedia.org/wiki/Comparison_of_shopping_cart_software) and [**CRM system**](https://en.wikipedia.org/wiki/Customer_relationship_management) system for your business. Platform contains basic internet shop functionality which allows user flexiable configure it. <br>

Everything is completely optimized even for virtual hosting, you can host this project on any **VDS/VPS** and **Virtual Hosting as well**. Frontend part optimized to interact on **Desktop** and **Mobile** devices

## Solution and Point of project
This project allow you to open up the E-commerce shop for your business. With wide functionality and as Monotlith app for low budget start if you wonder use virtual hosting instead separate VDS/VPS server.
Start your business in one click, that's the point what I'm working on.
